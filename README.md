# 📘 CBasic - Interactive C Programming Tutorial

CBasic is a beginner-friendly web application designed to help students learn the fundamentals of C programming through interactive lessons, code examples, and quizzes.

Built using pure HTML, CSS, and JavaScript, the project provides a simple learning environment without requiring any backend setup.

---

## ✨ Features

* Introduction to C programming
* Hello World example
* Variables and data types
* Loops (for and while)
* Functions
* Interactive code demonstrations
* Knowledge quiz
* Clean and beginner-friendly interface
