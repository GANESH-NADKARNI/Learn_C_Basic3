Why Learn C?

📦 Variables & Data Types

▼

Variables store data values. C has several basic data types:

Example: Variable Declaration 
#include <stdio.h>

int main() {
    int age = 25;
    float height = 5.9;
    char grade = 'A';
    
    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);
    printf("Grade: %c\n", grade);
    
    return 0;
}

🔄 Loops

                ▼
            

            

                
Loops allow you to repeat code multiple times. C has three main types of loops:

                
                

                    

                        Example: For Loop
                        
                    

                    
#include <stdio.h>

int main() {
    printf("Counting from 1 to 5:\n");
    
    for(int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    printf("\n");
    
    return 0;
}

⚙️ Functions

                ▼
            

            

                
Functions are reusable blocks of code that perform specific tasks:

                
                

                    

                        Example: Function Definition
                        
                    

                    
#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(10, 20);
    printf("10 + 20 = %d\n", result);
    
    printf("5 + 15 = %d\n", add(5, 15));
    
    return 0;
